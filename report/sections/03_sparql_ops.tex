\chapter{Opera\c{c}\~oes sobre os dados (SPARQL)}

\section{Vis\~ao geral da organiza\c{c}\~ao}

As opera\c{c}\~oes SPARQL do projeto foram reorganizadas por dom\'inio funcional na pasta \texttt{webapp/baseball/sparql\_queries/}. Esta divis\~ao permite separar claramente as queries de jogadores, equipas, ligas, grafos, vistas auxiliares e quiz.

\begin{itemize}[leftmargin=*]
  \item \texttt{players.py}: cat\'alogo de jogadores, filtros, detalhe biogr\'afico, hist\'orico desportivo, pr\'emios, All-Star Games, Hall of Fame e sal\'arios.
  \item \texttt{teams.py}: diret\'orio de franquias, hist\'orico por \'{e}poca, postseason, plant\'eis, managers, All-Stars e pr\'emios por equipa.
  \item \texttt{leagues.py}: navega\c{c}\~ao por liga, equipas da liga, s\'eries e l\'ideres estat\'isticos.
  \item \texttt{graphs.py}: queries usadas para construir os grafos relacionais dos detalhes de jogador e equipa.
  \item \texttt{misc.py}: listas auxiliares e rankings globais, como sal\'arios, Hall of Fame, managers e l\'ideres agregados.
  \item \texttt{quiz.py}: banco de perguntas do quiz e verifica\c{c}\~ao de respostas com \texttt{SELECT} e \texttt{ASK}.
  \item \texttt{edit\_service.py}: opera\c{c}\~oes \texttt{SPARQL UPDATE} para aprovar edi\c{c}\~oes sobre jogadores e equipas.
\end{itemize}

O ficheiro \texttt{webapp/baseball/sparql.py} continua apenas como superf\'icie de compatibilidade, reexportando as fun\c{c}\~oes dos m\'odulos internos.

\section{Queries por p\'agina}

Nesta sec\c{c}\~ao, as opera\c{c}\~oes s\~ao apresentadas por p\'agina ou contexto funcional da aplica\c{c}\~ao: rota ou contexto de utiliza\c{c}\~ao, fun\c{c}\~ao Python associada, descri\c{c}\~ao e tipo de opera\c{c}\~ao SPARQL. A listagem cobre as queries SPARQL de alto n\'ivel efetivamente usadas pelas p\'aginas e fluxos da aplica\c{c}\~ao. Fun\c{c}\~oes auxiliares que apenas agregam resultados em Python, como \texttt{get\_player\_batting\_summary()} e \texttt{get\_player\_pitching\_summary()}, n\~ao aparecem como entradas independentes porque n\~ao executam uma query adicional.

\subsection{Jogadores}

\begin{itemize}[leftmargin=*]
  \item \textbf{/players/ -- Contagem do cat\'alogo}
  
  Fun\c{c}\~ao: \texttt{get\_players\_catalog\_count()}
  
  Descri\c{c}\~ao: devolve o n\'umero total de jogadores que satisfazem os filtros ativos.
  
  Tipo SPARQL: \texttt{SELECT}

  SPARQL:

\begin{Verbatim}[fontsize=\scriptsize,breaklines=true]
PREFIX bb: <http://baseball.ws.pt/>
PREFIX foaf: <http://xmlns.com/foaf/0.1/>

SELECT (COUNT(DISTINCT ?player) AS ?total)
WHERE {
    ?player a bb:Player ;
            bb:playerID ?playerID ;
            foaf:name ?name .
    OPTIONAL { ?player bb:nameFirst ?nameFirst . }
    OPTIONAL { ?player bb:nameLast ?nameLast . }
    OPTIONAL { ?player bb:nameGiven ?nameGiven . }
    BIND(
        CONCAT(
            LCASE(COALESCE(STR(?name), "")), " ",
            LCASE(COALESCE(STR(?nameFirst), "")), " ",
            LCASE(COALESCE(STR(?nameLast), "")), " ",
            LCASE(COALESCE(STR(?nameGiven), ""))
        ) AS ?searchBlob
    )
}
\end{Verbatim}

  \item \textbf{/players/\{player\_id\}/ -- Resumo biogr\'afico e agregado}
  
  Fun\c{c}\~ao: \texttt{get\_player\_summary()}

  Tipo SPARQL: \texttt{SELECT}

  SPARQL:

\begin{Verbatim}[fontsize=\scriptsize,breaklines=true]
PREFIX bb: <http://baseball.ws.pt/>
PREFIX foaf: <http://xmlns.com/foaf/0.1/>

SELECT ?name ?nameFirst ?nameLast ?nameGiven ?playerID ?bbrefID
       ?birthCountry ?birthState ?birthCity ?bats ?throws ?debut ?finalGame
       ?awardsCount ?maxSalary
WHERE {
    ?player a bb:Player ;
            foaf:name ?name ;
            bb:playerID ?playerID .
    FILTER(?playerID = "aaronha01")

    OPTIONAL { ?player bb:bbrefID ?bbrefID . }
    OPTIONAL { ?player bb:nameFirst ?nameFirst . }
    OPTIONAL { ?player bb:nameLast ?nameLast . }
    OPTIONAL { ?player bb:nameGiven ?nameGiven . }
    OPTIONAL { ?player bb:birthCountry ?birthCountry . }
    OPTIONAL { ?player bb:birthState ?birthState . }
    OPTIONAL { ?player bb:birthCity ?birthCity . }
    OPTIONAL { ?player bb:bats ?bats . }
    OPTIONAL { ?player bb:throws ?throws . }
    OPTIONAL { ?player bb:debut ?debut . }
    OPTIONAL { ?player bb:finalGame ?finalGame . }
    OPTIONAL {
        SELECT (COUNT(DISTINCT ?awardObj) AS ?awardsCount)
        WHERE {
            ?player bb:playerID "aaronha01" ;
                    bb:wonAward ?awardObj .
        }
    }
    OPTIONAL {
        SELECT (MAX(?salaryValue) AS ?maxSalary)
        WHERE {
            ?player bb:playerID "aaronha01" ;
                    bb:hasSalary ?salaryObj .
            ?salaryObj bb:salary ?salaryValue .
        }
    }
}
LIMIT 1
\end{Verbatim}

\subsection{Equipas e franquias}

\begin{itemize}[leftmargin=*]
  \item \textbf{/teams/ -- Diret\'orio de franquias}
  
  Fun\c{c}\~ao: \texttt{get\_team\_franchise\_catalog()}
  
  Descri\c{c}\~ao: constr\'oi a grelha principal de equipas/franquias com informa\c{c}\~ao da \'{u}ltima \'{e}poca conhecida.
  
  Tipo SPARQL: \texttt{SELECT}

  SPARQL:

\begin{Verbatim}[fontsize=\scriptsize,breaklines=true]
PREFIX bb: <http://baseball.ws.pt/>

SELECT ?franchiseId ?franchiseName ?active
       ?year ?teamName ?teamId ?teamIdBr ?league ?division ?rank
       ?park ?attendance ?wins ?losses
WHERE {
    ?franchise a bb:Franchise ;
               bb:franchID ?franchiseId .
    OPTIONAL { ?franchise bb:franchiseName ?franchiseName . }
    OPTIONAL { ?franchise bb:active ?active . }

    ?team a bb:Team ;
          bb:franchiseOf ?franchise ;
          bb:yearID ?year .

    OPTIONAL { ?team bb:teamName ?teamName . }
    OPTIONAL { ?team bb:teamID ?teamId . }
    OPTIONAL { ?team bb:teamIDBR ?teamIdBr . }
    OPTIONAL { ?team bb:lgID ?league . }
    OPTIONAL { ?team bb:divID ?division . }
    OPTIONAL { ?team bb:Rank ?rank . }
    OPTIONAL { ?team bb:park ?park . }
    OPTIONAL { ?team bb:attendance ?attendance . }
    OPTIONAL { ?team bb:W ?wins . }
    OPTIONAL { ?team bb:L ?losses . }
}
ORDER BY ?franchiseId DESC(?year)
\end{Verbatim}

  \item \textbf{/teams/\{franchise\_id\}/ -- Hist\'orico sazonal da franquia}
  
  Fun\c{c}\~ao: \texttt{get\_team\_history()}
  
  Descri\c{c}\~ao: devolve as \'{e}pocas da franquia com informa\c{c}\~ao de contexto, recorde, batting, pitching e frequ\^encia.
  
  Tipo SPARQL: \texttt{SELECT}

  SPARQL:

\begin{Verbatim}[fontsize=\scriptsize,breaklines=true]
PREFIX bb: <http://baseball.ws.pt/>

SELECT ?year ?teamName ?teamId ?teamIdBr ?league ?division ?rank ?games ?homeGames
       ?wins ?losses ?runs ?attendance ?park
WHERE {
    ?franchise a bb:Franchise ;
               bb:franchID ?franchiseId .
    FILTER(?franchiseId = "ARI")

    ?team a bb:Team ;
          bb:franchiseOf ?franchise ;
          bb:yearID ?year .

    OPTIONAL { ?team bb:teamName ?teamName . }
    OPTIONAL { ?team bb:teamID ?teamId . }
    OPTIONAL { ?team bb:teamIDBR ?teamIdBr . }
    OPTIONAL { ?team bb:lgID ?league . }
    OPTIONAL { ?team bb:divID ?division . }
    OPTIONAL { ?team bb:Rank ?rank . }
    OPTIONAL { ?team bb:G ?games . }
    OPTIONAL { ?team bb:Ghome ?homeGames . }
    OPTIONAL { ?team bb:W ?wins . }
    OPTIONAL { ?team bb:L ?losses . }
    OPTIONAL { ?team bb:R ?runs . }
    OPTIONAL { ?team bb:attendance ?attendance . }
    OPTIONAL { ?team bb:park ?park . }
}
ORDER BY DESC(?year)
\end{Verbatim}

  \item \textbf{/teams/\{franchise\_id\}/ -- Hist\'orico de p\'os-temporada}
  
  Fun\c{c}\~ao: \texttt{get\_team\_postseason\_history()}
  
  Descri\c{c}\~ao: devolve as s\'eries de postseason em que a franquia participou, com resultado, advers\'ario e liga.
  
  Tipo SPARQL: \texttt{SELECT}

  SPARQL:

\begin{Verbatim}[fontsize=\scriptsize,breaklines=true]
PREFIX bb: <http://baseball.ws.pt/>

SELECT ?year ?round ?wins ?losses ?ties ?result ?teamName ?opponentName ?league ?opponentLeague
WHERE {
    ?franchise a bb:Franchise ;
               bb:franchID ?franchiseId .
    FILTER(?franchiseId = "ARI")

    ?series a bb:WorldSeriesResult ;
            bb:yearID ?year ;
            bb:round ?round ;
            bb:winnerTeam ?winnerTeam ;
            bb:loserTeam ?loserTeam .

    ?winnerTeam bb:franchiseOf ?winnerFranchise ;
                bb:teamName ?winnerTeamName .
    ?loserTeam bb:franchiseOf ?loserFranchise ;
               bb:teamName ?loserTeamName .

    FILTER(?winnerFranchise = ?franchise || ?loserFranchise = ?franchise)
}
ORDER BY DESC(?year) ?round
\end{Verbatim}

  \item \textbf{/teams/\{franchise\_id\}/ -- Plantel ofensivo da \'{e}poca}
  
  Fun\c{c}\~ao: \texttt{get\_team\_batting\_roster()}
  
  Descri\c{c}\~ao: devolve os jogadores com linhas de batting associadas \`a equipa numa determinada \'{e}poca.
  
  Tipo SPARQL: \texttt{SELECT}

  SPARQL:

\begin{Verbatim}[fontsize=\scriptsize,breaklines=true]
PREFIX bb: <http://baseball.ws.pt/>
PREFIX foaf: <http://xmlns.com/foaf/0.1/>

SELECT ?playerID ?name
       (SUM(?games) AS ?games)
       (SUM(?atBats) AS ?atBats)
       (SUM(?hits) AS ?hits)
       (SUM(?homeRuns) AS ?homeRuns)
WHERE {
    ?player a bb:Player ;
            bb:playerID ?playerID ;
            foaf:name ?name ;
            bb:hasBatting ?batting .
    ?batting bb:teamOf ?team .
    ?team bb:teamID ?teamId ;
          bb:yearID ?teamYear .
    FILTER(?teamId = "ARI" && ?teamYear = 2015)

    OPTIONAL { ?batting bb:G ?games . }
    OPTIONAL { ?batting bb:AB ?atBats . }
    OPTIONAL { ?batting bb:H ?hits . }
    OPTIONAL { ?batting bb:HR ?homeRuns . }
}
GROUP BY ?playerID ?name
ORDER BY DESC(?atBats) DESC(?homeRuns) ?name
\end{Verbatim}

    OPTIONAL { ?pitching bb:SO ?strikeouts . }

  \item \textbf{/teams/\{franchise\_id\}/ -- Managers da \'{e}poca}
  
  Fun\c{c}\~ao: \texttt{get\_team\_managers()}
  
  Descri\c{c}\~ao: devolve os managers associados \`a equipa e os respetivos dados b\'asicos de desempenho.

WHERE {
    ?player a bb:Player ;
            bb:playerID ?playerID ;
            bb:isManager ?manager .

    OPTIONAL { ?manager bb:losses ?losses . }
    OPTIONAL { ?manager bb:rank ?rank . }
    OPTIONAL { ?manager bb:inseason ?inseason . }
}
ORDER BY ?inseason ?rank ?name
\end{Verbatim}

  Fun\c{c}\~ao: \texttt{get\_team\_all\_stars()}
  
  Descri\c{c}\~ao: devolve os jogadores da equipa que participaram em All-Star Games nessa \'{e}poca.
  
  Tipo SPARQL: \texttt{SELECT}

  SPARQL:
